## 1) T-shirt (DTF Junkie + Ton Atelier)
### 1.1 DTF Junkie - T-shirt Placement Guide (source-exact -> cm)
- Centered design placement: `7.6-8.9 cm from collar`
- Centered design size:
  - Adult: `27.9 x 27.9 cm`
  - Youth (med/lg): `26.7 x 26.7 cm`
  - Youth (small): `21.6 x 21.6 cm`
  - Toddler: `14.0 x 14.0 cm`
- Left chest placement: `14.0-20.3 cm down from shoulder left seam`, `10.2-15.2 cm over from center`
- Left chest size:
  - Adult: `8.9 x 8.9 cm`
  - Youth (med/lg): `8.9 x 8.9 cm`
  - Youth (small): `8.9 x 8.9 cm`
  - Toddler: `6.4 x 6.4 cm`

### 1.2 DTF Junkie - T-shirt Visual Quick Guide (source-exact labels -> cm)
- Max imprint area (full front/back): `30.5 x 35.6 cm`
- Medium front/back imprint area: `20.3 x 20.3 cm`
- Across chest imprint area: `10.2 x 30.5 cm`
- Left/Right chest imprint area: `8.9 x 8.9 cm` (chart deer `12.7 x 12.7 cm` variant ch baina)
- Sleeve imprint area: `12.7 x 35.6 cm` / `10.2 x 35.6 cm` (chart variant)
- Locker patch area: `10.2 x 10.2 cm` (chart label)

### 1.3 Ton Atelier - T-shirt Printing Placement and Sizing (source-exact, medium tee -> cm)
- Front L chest print: width `7.6-10.2 cm`, placement `7.6 cm from collar`, example `10.2w x 7.6h cm`
- Front chest print: width `20.3-25.4 cm`, placement `7.6 cm from collar`, example `25.4w x 5.1h cm`
- Front full print: width `25.4-30.5 cm`, placement `5.1 cm from collar`, example `30.5w x 30.5h cm`
- Sleeve print: width `7.6 cm`, placement `10.2 cm from collar`, example `7.6w x 7.6h cm`
- Neck tag: width `5.1 cm`, placement `2.5 cm from collar`, example `5.1w x 5.1h cm`
- Back print: width `30.5 cm`, placement `10.2 cm from collar`, example `30.5w x 35.6h cm`

## 4) Admin size mapping (inference removed from numeric values)
Doorh map ni hemjeesiin too-g oorchluhgui, zuvhun source category-g size keys ruu holboj baigaa.

### 4.1 T-shirt / Hoodie category mapping
- Adult sizes: `XS, S, M, L, XL, 2XL, 3XL` -> `Adult`
- Youth sizes: `Youth XS, Youth S` -> `Youth (small)`
- Youth sizes: `Youth M, Youth L, Youth XL` -> `Youth (med/lg)`
- Toddler sizes: `2T, 3T, 4T, 5T` -> `Toddler`

### 4.2 Jersey category mapping
- Adult sizes: `XS, S, M, L, XL, 2XL, 3XL` -> `Adult`
- Youth sizes: `Youth XS, Youth S, Youth M, Youth L, Youth XL` -> `Youth`
